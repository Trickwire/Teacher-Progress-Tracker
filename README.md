# Teacher Progress Tracker

A simple, opinionated tool that turns teacher observations into clean, printable progress sheets—no Excel, no complexity, no design decisions.

## What It Does

- **Guided Input**: Simple form with predefined observation types
- **Automatic Organization**: Groups observations by student automatically
- **Printable Sheets**: One-click printing with professional formatting
- **Automatic Summaries**: Counts and statistics generated automatically
- **No Setup**: Works immediately, no configuration needed

## How to Use

1. **Open** `index.html` in any web browser
2. **Add Observations**: Fill in the form with student name, date, type, and notes
3. **View Progress**: See all observations organized by student below
4. **Print**: Click "Print Selected" to generate clean progress sheets
5. **Filter**: Use the dropdown to view a specific student's progress

## Observation Types

- **Academic Progress**: Learning achievements and academic milestones
- **Behavior**: Behavioral observations and conduct notes
- **Participation**: Engagement and participation in class
- **Social Skills**: Interactions with peers and social development
- **Strength/Interest**: Student strengths and interests
- **Area of Concern**: Issues or concerns that need attention

## Features

- ✅ All data stored locally in your browser (no server needed)
- ✅ Works offline after first load
- ✅ Print-optimized formatting
- ✅ Automatic date formatting
- ✅ Summary statistics for each student
- ✅ Filter by student name
- ✅ No Excel required

## Technical Details

- Pure HTML, CSS, and JavaScript
- Uses browser localStorage for data persistence
- No dependencies or build process required
- Works in all modern browsers

## Privacy

All data is stored locally in your browser. Nothing is sent to any server. Your observations stay on your computer.
